* Introduction 
  This document describes the testing done on
  =vlabs-landing-pages=.

* Test Environment
  To Test the =vlabs-landing-pages=  the following servers
  have been used.
  1) =OpenedX= Ginkgo version
  2) =LabDataService= Release =v3.2.1= version hosted on
     base4 with URL: [[http://lds-alpha.base4.vlabs.ac.in/institutes][http://lds-alpha.base4.vlabs.ac.in/institutes]]
  3) =VlabsLandingPages= branch =abinitio= has been setup on
     local machine with the steps mentioned in the [[https://github.com/vlead/vlabs-landing-pages/blob/master/src/deployment/setup-dev-environment.org][set up document]]
     
* Procedure
  Testing of =vlabs-landing-pages= is done manually. The
  following sections have been tested and issues has raised
  against each lab repository on github.
  1) Lab URL (Working/Not Working)
  2) Experiment URL (Working/Not Working)
  3) Full Screen Tab in case of OpenedX labs (Working/Not
     Working)
  4) Feedback Links and
  5) Analytics Usage
